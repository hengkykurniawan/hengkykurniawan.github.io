{%- comment -%} Client-side MathJax fallback. Renders $...$ (inline) and $$...$$ (display) math directly in the browser. This guarantees equations show up even if the jekyll-spaceship plugin is not run at build time (e.g. on GitHub Pages' classic/safe-mode build). If jekyll-spaceship HAS already rendered the math at build time, no literal $...$ delimiters remain in the page, so MathJax simply finds nothing to do — no double rendering. {%- endcomment -%}
